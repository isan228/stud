# Платформа для студентов

Веб-приложение для подготовки студентов к тестам и экзаменам.

## Технологии

- **Backend**: Node.js, Express.js
- **База данных**: PostgreSQL
- **ORM**: Sequelize
- **Frontend**: EJS, Bootstrap 5
- **Аутентификация**: Express-session

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Убедитесь, что PostgreSQL запущен и создана база данных `studd` с паролем `12345678`

3. Синхронизируйте базу данных:
```bash
npm run sync-db
```

4. Запустите сервер:
```bash
npm start
```

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

## Настройка базы данных

- Имя БД: `studd`
- Пользователь: `postgres`
- Пароль: `12345678`
- Хост: `localhost`

## Структура проекта

```
├── config/          # Конфигурация базы данных
├── models/          # Модели Sequelize
├── routes/          # Маршруты Express
├── views/           # EJS шаблоны
├── public/          # Статические файлы (CSS, JS)
├── server.js        # Главный файл сервера
└── sync-db.js      # Скрипт синхронизации БД
```

## Функционал

- Регистрация и авторизация пользователей
- Личный кабинет с возможностью смены пароля
- Эмулятор тестов с настройками
- Статистика прохождения тестов
- Система подписок
- Уведомления об ошибках в тестах
